# Flutter 杰理OTA升级(Flutter)


杰理OTA升级APP是⼀款专为使⽤杰理芯⽚的设备设计的在线升级⼯具。它允许⽤户通过蓝⽛对设备进⾏固件升
级，以确保设备始终拥有最新的功能和安全修复。

|              | Android | iOS   |
| ------------ | ------- | ----- |
| **版本支持** | SDK 23+ | 12.0+ |

---


## 插件引用

```pubspec.yaml
    plugin:
    platforms:
      android:
        package: com.jieli.otasdk
        pluginClass: JlOtaPlugin
      ios:
        pluginClass: JlOtaPlugin
```
